# Pasta Public

Coloque aqui arquivos estáticos que serão servidos na raiz do site.

## Foto do Especialista

Para adicionar a foto do especialista na seção "Sobre", adicione uma imagem com o nome:

**`especialista.jpg`**

Ou altere o caminho no componente `About.jsx` para o nome do seu arquivo.

Formatos suportados: JPG, PNG, WebP
